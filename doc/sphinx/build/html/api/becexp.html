Convert absorption signal to optical depth (OD).

computeAbsorption(imageData, absorpMin)#

CALCULATEABSORPTION calculate absorption signal imageData: must be a 4-D array. The first two dimensions store the image data. The third dimension is the run index. The last dimension represents atom/light/dark image labels (or atom/light). absorp: absorption signal. Must be between 0 and 1 where 0 means totally absorbed by atoms.
